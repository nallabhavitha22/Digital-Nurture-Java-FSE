How to Use the Portal

Follow these simple instructions to use the portal smoothly.

Use the top navigation bar to move between Home, Events, Register, Feedback, Location, and Contact sections.

Registration Fill in your name, email, phone, event date, event type, and message.
Gallery Double-click an event image to enlarge or reduce it.
Location Click Find Nearby Events and allow location permission.

To inspect the page structure, open Chrome, right-click the page, and select Inspect. Use the Elements tab for HTML/CSS and the Console tab for JavaScript logs.
